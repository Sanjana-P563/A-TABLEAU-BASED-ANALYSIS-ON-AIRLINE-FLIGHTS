# A-TABLEAU-BASED-ANALYSIS-ON-AIRLINE-FLIGHTS
# Evaluated a large airline flights dataset (300,000+ records) using Tableau to study pricing trends, flight duration, stops, and scheduling patterns between major cities.
# Executed data cleaning and preprocessing by removing unnecessary columns,and preparing data for effective visualization in Tableau.
# Created interactive visualizations (bar charts, pie charts, bubble charts) to compare airlines on metrics such as average ticket price, flight duration, number of stops, and departure/arrival times.
# Designed an interactive Tableau dashboard to deliver actionable insights on price variation by airline, booking time, and class, supporting data-driven decision-making for travelers and analysts.
